# 유튜브 다운로더

유튜브 영상을 검색하고 다운로드할 수 있는 웹 애플리케이션입니다.

## 기능

- 🔍 유튜브 영상 검색 (제목, 링크, 내용)
- 📥 다운로드 (음악, 영상, 자막)
- ⭐ 찜하기 기능
- 📁 찜하기 폴더 관리 (재생목록처럼)

## 설치

```bash
pip install -r requirements.txt
```

## 실행

```bash
python app.py
```

브라우저에서 `http://localhost:5000`으로 접속하세요.

## 사용법

1. 검색창에 유튜브 영상 제목, 링크, 또는 내용을 입력
2. 검색 결과에서 원하는 영상 선택
3. 다운로드 형식 선택 (음악/영상/자막)
4. 찜하기 버튼으로 나중에 볼 영상 저장
5. 찜하기 폴더를 만들어서 영상 관리
