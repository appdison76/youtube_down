# 스크린샷 이미지 안내

이 폴더에 앱 스크린샷 이미지를 넣어주세요.

## 필요한 이미지 파일

다음 3개의 스크린샷 이미지를 이 폴더에 넣어주세요:

1. **screenshot-search.png** (또는 .jpg)
   - 검색 화면 스크린샷
   - 권장 크기: 최소 600x1000px (세로 비율)
   - 용량: 최대 500KB 권장

2. **screenshot-download.png** (또는 .jpg)
   - 다운로드 화면 스크린샷 (SearchScreen)
   - 권장 크기: 최소 600x1000px (세로 비율)
   - 용량: 최대 500KB 권장

3. **screenshot-files.png** (또는 .jpg)
   - 파일 관리 화면 스크린샷 (DownloadsScreen)
   - 권장 크기: 최소 600x1000px (세로 비율)
   - 용량: 최대 500KB 권장

## 파일명 변경 시

다른 파일명을 사용하고 싶다면 `index.html` 파일에서 이미지 경로를 수정해주세요.

예:
- `screenshots/my-search.png`
- `screenshots/my-download.png`
- `screenshots/my-files.png`

## 이미지 최적화 팁

- PNG 또는 JPG 형식 사용
- 모바일 화면 비율 (세로) 권장
- 웹용으로 최적화 (용량 최소화)
- 고해상도 권장 (Retina 디스플레이 대응)
