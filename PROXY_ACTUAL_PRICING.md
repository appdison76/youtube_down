# 프록시 서비스 실제 가격 (2024년 확인)

## ⚠️ 중요: 대부분의 프록시 서비스는 GB 단위 과금

대부분의 프록시 서비스는 **"무제한"** 플랜이 없거나 매우 비쌉니다.
대신 **GB 단위로 과금**하는 모델을 사용합니다.

---

## 💰 실제 가격 (2024년 확인)

### Smartproxy (Decodo)

#### GB 단위 과금 (표준)
- **2GB**: $6/월 (50% 할인 코드 사용 시)
- **8GB**: $22/월
- **25GB**: $65/월
- **100GB**: $225/월
- **1000GB**: $1,500/월
- **Pay-as-you-go**: $3.5/GB

#### 무제한 플랜
- 별도 문의 필요
- 동시 연결 수(concurrency) 기준으로 과금될 수 있음
- $75/월에 무제한인지는 확인 필요

**확인 방법**: https://smartproxy.com/proxies/residential-proxies/pricing

---

### IPRoyal

- **1GB**: $3.50/월 (Subscription)
- **2GB**: $5.95/월
- **10GB**: $52.50/월 ($5.25/GB)
- **50GB**: $245/월 ($4.90/GB)

#### 무제한 플랜
- 별도 문의 필요
- $75/월에 무제한인지는 확인 필요

**확인 방법**: https://iproyal.com/pricing/residential-proxies/

---

- **Enterprise**: 맞춤형

**확인 방법**: https://oxylabs.io

### Bright Data

## ⚠️ 주의사항

1. **$75/월 무제한은 확인 필요**: 대부분의 서비스는 GB 단위 과금
2. **실제 가격 확인**: 각 서비스 웹사이트에서 최신 가격 확인
3. **할인 코드**: Smartproxy는 "RESI50" 코드로 50% 할인 가능
4. **무제한 플랜**: 별도 문의 필요할 수 있음

---

## 📞 확인 방법

1. 각 서비스 웹사이트 방문
2. 가격 페이지 확인
3. 무제한 플랜 존재 여부 확인
4. 필요시 영업팀에 문의
